# Ian Roblox Learning Buddy Reference

This folder is the canonical visual reference pack for Ian's learning buddy.

Use this pack whenever creating new worksheet scenes, avatar overlays, reward
art, cover art, or other Ian-specific visual content.

## Required Likeness

- Roblox-inspired blocky boy avatar in a 2D comic/game-instruction style.
- Square peach head with rounded corners.
- Simple black oval eyes and friendly curved smile.
- Vivid rainbow spiky side-swept hair with red, orange, yellow, green, cyan,
  blue, purple, and magenta sections.
- Bright blue short-sleeve shirt with a large yellow lightning bolt.
- Peach rectangular block arms and hands.
- Brown straight blocky pants.
- Orange sneakers with white soles and white lace details.
- Optional medium-brown backpack when the scene calls for a school/task prop.
- Bold black outlines, saturated colors, and calm light-blue/home/obby settings.

Do not change the hair, face, blue lightning shirt, brown pants, orange shoes,
or blocky proportions. Accessories must not cover the hair, face, lightning
bolt, pants, or shoes.

## Scene Rule

Worksheet art should be a full activity scene, not a static avatar sticker or a
crop from this reference sheet. Use the reference images only to preserve Ian's
likeness, then create new art where Ian is doing the page task. Add a small prop
set tied to the literacy objective: picture cards for matching, letter blocks or
writing lines for word work, a book/card for fluency, sound-wave cues for
listening, and celebration only for reward/cover moments.

Activity content images should be attractive standalone pictures of the target
word or concept. Ian does not need to appear inside every matching tile; those
images should help the child understand the word.
